<h1 align="center">Dictator</h1>

<p align="center">
  Голосовой ввод текста для Windows через Yandex SpeechKit
</p>

<p align="center">
  <a href="../../releases/latest"><strong>Скачать последнюю версию</strong></a>

---

**Dictator** — бесплатная программа для голосового набора текста на Windows. Нажмите горячую клавишу, продиктуйте текст, и он автоматически скопируется в буфер обмена. Вставляйте через Ctrl+V в любое приложение: мессенджер, редактор, браузер.

Распознавание речи выполняется через [Yandex SpeechKit](https://yandex.cloud/ru/services/speechkit) — точное распознавание русской речи, работающее быстро и надёжно.

## Возможности

- **Голосовой ввод в любое приложение** — результат копируется в буфер обмена автоматически
- **Глобальная горячая клавиша** — начинайте диктовку из любого окна (по умолчанию Ctrl+Shift+R)
- **Настраиваемое сочетание клавиш** — задайте удобную комбинацию в настройках
- **Системный трей** — приложение работает в фоне и не мешает
- **История распознаваний** — последние 30 записей с возможностью копирования
- **Один EXE-файл** — не требует установки, .NET Runtime или дополнительных зависимостей
- **Шифрование API-ключа** — ключ хранится локально, зашифрован Windows DPAPI
- **Тёмная тема** — современный интерфейс с Mica/Acrylic эффектами

## Скачать

Перейдите во вкладку **[Releases](../../releases)** и скачайте `Dictator.exe` из последнего релиза. Один файл, установка не нужна.

> Требуется Windows 10 (1809) или новее. x64.

## Быстрый старт

1. Запустите `Dictator.exe`
2. Откройте настройки (шестерёнка в заголовке окна)
3. Вставьте API-ключ Yandex Cloud ([как получить](#получение-api-ключа))
4. Нажмите **Ctrl+Shift+R** (или свою горячую клавишу) и говорите
5. Текст автоматически скопируется в буфер — вставьте через **Ctrl+V**

## Получение API-ключа

1. Зайдите на [aistudio.yandex.ru](https://aistudio.yandex.ru)
2. Создайте API-ключ в разделе ключей
3. Вставьте ключ в настройках приложения

Yandex SpeechKit предоставляет бесплатный лимит на распознавание речи.

## Сборка из исходников

```bash
git clone https://github.com/luxygolf/YandexDictator.git
cd YandexDictator
dotnet publish -c Release
```

Результат: `bin/Release/net8.0-windows10.0.19041.0/win-x64/publish/Dictator.exe`

### Требования для сборки

- .NET 8 SDK
- Windows 10/11
- Visual Studio 2022+ с компонентом "Windows App development" (для WinUI 3)

## Технологии

- [WinUI 3](https://learn.microsoft.com/windows/apps/winui/winui3/) — современный UI-фреймворк Windows
- [Yandex SpeechKit](https://yandex.cloud/ru/services/speechkit) — распознавание русской речи
- [NAudio](https://github.com/naudio/NAudio) — запись с микрофона
- [H.NotifyIcon](https://github.com/HavenDV/H.NotifyIcon) — иконка в системном трее

## Конфиденциальность

- Аудиозаписи отправляются на серверы Yandex Cloud для распознавания
- API-ключ хранится локально, зашифрован Windows DPAPI
- История распознаваний хранится локально в `%APPDATA%/Dictator/`
- Приложение не собирает и не передаёт никаких других данных
